  • <!DOCTYPE> - code language

  • <head> - head

  • <title> - page title

  • <footer> - page footer

  • <h1> to <h6> - headings

  • <p> - normal

  • <img> - image

  • <a> - hyperlink


  • <b> - bold

  • <i> - italic

  • <q> - quotes

  • <del> - strikethrough

  • <ins> - underline


  • <em> - normal emphasize

  • <strong> - normal strong

  • <mark> - normal mark

  • <sub> - normal subscript

  • <sup> - normal superscript


  • <ul> - unordered list

  • <ol> - ordered list

  • <li> - list item
